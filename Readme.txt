How to use :

- Put the folder file in the path where you installed the node js
- Run npm start from terminal to access the website
- In home page, click browse movies
- Choose the movie that you want and click detail to see information of the movie.

Description of project :

- This project contain information from Star Wars API (https://swapi.co/)
- This web contain 4 main pages i.e, home page, film page, detail page, and error page.
- Home page is displaying the landing page.
- Film page is displaying star wars movies list from episode 1-7
- Detail page is displaying the detail information of the movie like release date, title, characters, planets, starships, etc.
- And the error page just displayed when there is no page found.
